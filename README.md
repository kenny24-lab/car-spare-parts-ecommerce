# Car Spare Parts E-Commerce Website

## Project Overview

The Car Spare Parts E-Commerce Website is a web-based platform that allows customers to browse, search, and purchase genuine car spare parts online. The system provides a user-friendly interface for customers and an administration panel for managing products, categories, orders, and users.

## Installation Guide

### Requirements

- PHP 8.x or later
- MySQL Database
- Apache Server (XAMPP, WAMP, or Hosting Server)

### Steps

1. Clone the repository

```bash
git clone https://github.com/yourusername/car-spare-parts-ecommerce.git
```

2. Move the project to your web server directory.

3. Create a MySQL database.

4. Import the provided SQL file into the database.

5. Update database credentials in:

```php
config/database.php
```

Example:

```php
$host = "your_host";
$username = "your_username";
$password = "your_password";
$database = "your_database";
```

6. Start Apache and MySQL.

7. Open the project in your browser:

```text
http://localhost/car-spare-parts-ecommerce
```
